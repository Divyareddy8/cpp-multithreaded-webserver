Multi-threaded · LRU Cache · Thread Pool · RAII Sockets

⚡

Thread Pool

Fixed worker threads pre-allocated at startup. Zero spawn overhead per request — tasks are queued and dispatched with mutex + condition_variable.

🗄️

LRU Cache

32MB in-memory Least Recently Used cache for static files. Thread-safe with its own mutex. Cache hits skip disk I/O entirely.

🔒

RAII Sockets

Every client socket is wrapped in a SocketGuard. Zero file descriptor leaks guaranteed — destructor closes the fd automatically.

🛡️

Path Traversal Guard

All requested paths are canonicalized and verified to be inside the web root. ../../etc/passwd attacks return 403.
